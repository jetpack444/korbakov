JavaScript Практика - 24 задания

1. Длина имени

Введите свое имя "yourName", посчитайте его длину с помощью свойства "yourName".length

2. Сложение чисел

Выполните операцию сложения с цифрами 3, 5.

3. Деление с confirm

Выполните операцию деления с цифрами 16 и 2. Сделайте вывод частного с помощью функции диалогового окна confirm().

4. Возраст в консоль

Задайте свой возраст с помощью переменной age. Выведите значение переменной в консоль браузера с помощью функции console.log().

5. Ввод слова и обработка

Сделайте ввод любого слова с помощью функции prompt(). Установите его длину, выведете значение длины в консоль. С помощью свойства substring() выведете в консоль первые две буквы слова.

6. Сравнение возраста родителей

Запросите у пользователя с помощью функции prompt() возраст его родителей. И напишите условие, при котором выводится имя того, кто старше. Используем оператор условия if.

7. Концерт Аллы Пугачевой

Ребенок желает пойти на выступление известных артистов. В городе проводится концерт только Аллы Пугачевой, куда пускают детей старше 12 лет. Необходимо уточнить возраст ребенка и спросить готов ли он отправиться слушать эту певицу. При утвердительном ответе вывести «Вперед за билетами», при отрицательном «Сиди дома».

8. Функция возведения в квадрат

Напишите функцию возведения любого числа в квадрат, результат вычисления выведете в диалоговое окно, используя структуру функции function(){};.

9. Функция приветствия

Напишите функцию приветствия. Используя ее, пользователь задает свое имя, в итоге в диалоговое окно должно быть выведено "Доброго здоровечка Имя_Пользователя".

10. Проверка деления на 4

Написать функцию, которая возвращает частное от деления на 4. Ввести через диалоговое окно переменную и выполнить сравнение, если остаток от деления (оператор %) равен нулю, то выводим в диалоговое окно «Выражение истинно», в противном случае «Выражение ложно».

11. Периметр прямоугольника

Напишите функцию расчета периметра прямоугольника function(a,b){};

12. Цикл for 1-10

Сделайте вывод в консоль цифры от 1 до 10, используя цикл for.

13. Цикл for 5-50 с шагом 5

Сделайте вывод в консоль цифр от 5 до 50 с шагом 5, используя цикл for.

14. Цикл while 1-10

Выполните упражнение 12, используя цикл while.

15. Цикл while 5-50 с шагом 5

Выполните упражнение 13, используя цикл while.

16. Цикл с сообщением

Пусть переменная i=5, напишите цикл, в котором, при условии, что i<1, хотя бы раз выведется сообщение «Нас не задушишь, не убьешь».

17. Меню с switch

Перед вами меню, в котором можно выбрать: яблоко, апельсин, сок, пиво. С помощью диалогового окна, введите то, что вы желаете заказать. С помощью switch() на ваш выбор в консоли отобразиться фраза: «Ваш выбор (название) + просто идеален». При наборе слова, которого нет в меню, выводится фраза «Пошел прочь!»

18. Массив с 5 элементами

Создайте массив, используя литеры [], заполните его пятью элементами, выведите на экран значение третьего элемента.

19. Длина массива

Выведите на экран значение длины массива пункта 18, используя свойство length.

20. Все элементы массива

Выведите на экран все значения элементов массива пункта 18.

21. Массив через конструктор

Создайте пустой массив, используя конструктор Array, наполните его элементами, один из элементов должен быть массивом. Выведите элементы массива на экран.

22. Объект телефонная книга

Создайте объект телефонная книга, используя литеры {}, добавьте в него свойства ФИО, и телефонный номер. Выведите значения свойств объекта на экран.

23. Объект через конструктор

Создайте любой объект с пятью свойствами, используя конструктор Object. Выведите все свойства, используя цикл for in.

24. Значения свойств объекта

Используя, цикл for in выведите на экран значения свойств объекта пункта 23.
