Контрольные вопросы

1. Чем отличается display: flex от display: grid?
Display: flex и display: grid — это два инструмента в CSS, которые используются для создания гибких и адаптивных макетов. Они имеют разные подходы и возможности.
Flexbox (или Flexible Box Layout) предназначен для создания одномерных макетов, где элементы располагаются в одном направлении (горизонтально или вертикально).
Элементы могут изменять свою величину, чтобы заполнять доступное пространство. 13
CSS Grid предназначен для создания двумерных макетов, где элементы располагаются в двух направлениях (горизонтально и вертикально). Позволяет создавать сложные сетки с
явным контролем за размещением элементов в ячейках
